Bem vindo a documentação do projeto TCC-SPTECH-CCO-2023 do Grupo 15
===================================================================

O TCC-SPTECH-CCO-2023 é um projeto de conclusão de curso do grupo 15 do curso de Ciência da Computação da Faculdade SPTech Digital School.

O projeto consiste em uma aplicação de machine learning e IA para a detecção de anomalias em radiografias de tórax.

O projeto foi desenvolvido em Python e utiliza majoritariamente as bibliotecas PyTorch, Pandas, Numpy.
